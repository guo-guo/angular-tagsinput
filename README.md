# angular-tagsinput
使用angularJS的directive编写tags和普通输入完美结合。使用了typeahead作为自动提示。

可以指定相同的ng-model使多个输入框中数据同步。

example:

`<div class="bootstrap-tagsinput" customtags ng-model="query.querySearch" >`
